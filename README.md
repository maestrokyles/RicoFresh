# RicoFresh
Your pocket Leetcode Flashcard - mobile app

## 0. Overview
This is an iOS mobile app for tech job seekers to manage code challenges. There are four modes: 
- Problem manager
- Flashcard
- Daily Challenge
- Problem CRUD


## 1. Tech Stack
Language
- Javascript

Database:
- Google Firebase

Libraries: 
- React Native
- react-native-gesture-handler
- react-native-ico-material-design

Dev tool:
- Expo
- Visual studio code
